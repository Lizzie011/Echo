## 遇到的问题

1. 分类名称/颜色编辑后，列表页不刷新，需要重启 App 才显示
   - 根因：ArkTS V2 中 ForEach + key 函数复用 DOM 节点时，`@Builder` 函数不会重新执行
   - 解决：将行内容从 `@Builder buildCategoryRow(cat)` 改为直接内联在 ForEach 回调中
   - 模式对比：
     ```
     // ❌ ForEach + key 复用时，@Builder 不重新执行
     ForEach(arr, item => this.buildRow(item), key: item => item.id)
     
     // ✅ 内联渲染，ForEach 每次迭代都执行最新代码
     ForEach(arr, item => { /* inline Row() { ... } */ }, key: item => item.id)
     ```
