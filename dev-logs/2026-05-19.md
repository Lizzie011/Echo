## 数据流架构（最终版）

```
fullClips（主列表，全量数据）
    ↓ applyFilters()
    ↓ 根据 searchKeyword / selectedCategoryId 过滤
    ↓ pinned 优先排序
    ↓
clips（@Trace，UI 展示列表）
    ↓
Scroll + Column + ForEach (key=id) 渲染
```

**所有写操作**: 更新 DB + 更新 fullClips → applyFilters()
**搜索/筛选**: 设置 searchKeyword/selectedCategoryId → applyFilters()
**loadClips**: 仅在 init() 时调用，加载 DB 数据到 fullClips
