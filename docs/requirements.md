# Echo — 需求文档

## 项目概述

Echo 是一款 HarmonyOS 历史剪贴板管理应用。用户复制文字或图片后，在 Echo 中可查看、搜索、分类管理所有历史剪贴内容。

## 核心功能

| 编号 | 功能 | 说明 |
|------|------|------|
| F1 | 自动记录剪贴板 | App 在前台时监听系统剪贴板变化，自动记录文字和图片 |
| F2 | 首页时间线 | 所有内容按时间倒序卡片排列，最新在上 |
| F3 | 点击再次复制 | 点击卡片将内容写回系统剪贴板，显示"已复制!"提示 |
| F4 | 搜索 | 顶部搜索栏，对文字内容关键词模糊搜索 |
| F5 | 置顶/取消置顶 | 长按卡片选择置顶，置顶内容始终排在最前 |
| F6 | 删除 | 左滑直接删除，或长按选择删除 |
| F7 | 分类管理 | 创建/重命名/删除分类，自定义颜色 |
| F8 | 移动到分类 | 长按卡片→选择"移动到..."→选择目标分类 |
| F9 | 存储时效 | 全局设置保留天数（1/3/5/7天/永不），过期自动清理 |
| F10 | 分类筛选 | 横滑标签栏按分类查看内容 |
| F11 | 清空所有 | 一键清空全部记录 |

## 功能介绍

| 项目 | 决定 |
|------|------|
| 剪贴板监控方式 | 前台监控 — App 打开时记录复制内容 |
| 分类方式 | 一对一 — 每条内容只属于一个分类 |
| 存储时效 | 全局统一 — 一个设置控制所有内容过期时间 |

## UI 结构

3 个底部 Tab：
- **Tab1 - 剪贴板**：搜索栏 + 分类筛选标签 + 内容卡片列表
- **Tab2 - 分类管理**：分类列表 + 新建/编辑/删除
- **Tab3 - 设置**：存储时效选择器 + 清空所有 + 关于

## 数据模型

### 分类表 (categories)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| name | TEXT | 分类名称 |
| color | TEXT | 分类颜色标识 |
| createdAt | INTEGER | 创建时间戳 |

### 剪贴内容表 (clips)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| type | TEXT | 'text' 或 'image' |
| content | TEXT | 文字内容或图片文件路径 |
| categoryId | INTEGER FK | 所属分类ID |
| isPinned | INTEGER | 0=否 1=是 |
| createdAt | INTEGER | 创建时间戳 |
| expiresAt | INTEGER | 过期时间戳（0=永不过期） |
