# Echo — 设计规范

## 色彩系统

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色调 (Accent) | `#FF6B6B` | 按钮、标签、选中状态、品牌色 |
| 主背景 | `#F8F8F8` | 页面底色 |
| 卡片背景 | `#FFFFFF` | 内容卡片 |
| 主文字 | `#1A1A1A` | 标题、正文 |
| 次要文字 | `#999999` | 时间戳、提示 |
| 辅助文字 | `#BBBBBB` | 占位引导 |
| 分隔线 | `#F0F0F0` | 卡片描边、分割 |
| 输入框背景 | `#F5F5F5` | 搜索栏、输入框 |
| 删除色 | `#FF4444` | 删除按钮、确认删除 |
| 遮罩 | `rgba(0,0,0,0.4)` | 弹窗遮罩 |

## 分类预设颜色

`#FF6B6B` `#4ECDC4` `#FFD93D` `#6C5CE7` `#A8E6CF` `#FF8A80` `#80D8FF` `#FFB74D`

## 组件规范

### 卡片
- 圆角: 14px
- 描边: 1px `#F0F0F0`
- 内边距: 14px
- 间距: 上下 4px

### 按钮 / 标签
- 圆角: 16-20px
- 选中态: 主题色填充 + 白色文字
- 默认态: `#EEEEEE` 填充 + `#666666` 文字

### 输入框
- 圆角: 10-20px
- 背景: `#F0F0F0` 或 `#F5F5F5`

## 图标

全部使用 Emoji，不依赖外部图标资源：
- 📋 剪贴板 Tab
- 📂 分类 Tab
- ⚙ 设置 Tab
- 🔍 搜索
- 📌 置顶
- 🗑 删除
- 📝 文字类型
- 🖼 图片类型

## 交互规范

- **点击卡片** → 复制内容到剪贴板，弹出"已复制!"提示
- **左滑卡片** → 显示删除按钮（红色背景）
- **长按卡片** → 弹出操作面板（置顶 / 移动到 / 删除）
- **点击分类标签** → 筛选该分类内容
- **弹窗** → 居中模态框 + 半透明遮罩，点击遮罩关闭

## 命名规范

- 文件名: PascalCase（如 `ClipCard.ets`）
- 变量/方法: camelCase（如 `loadClips`）
- 常量: UPPER_SNAKE_CASE（如 `DEFAULT_CATEGORY_NAME`）
- 类名: PascalCase（如 `ClipboardViewModel`）
- 组件参数避免与系统属性冲突（如不用 `clip`，改用 `clipItem`）
